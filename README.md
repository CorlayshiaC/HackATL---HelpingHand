# HackATL— Server
  Team: MysteryGang - Cor'layshia, Jacob, Shailah
  
A community-driven platform to create, support, and track campaigns for social good. Built with Deno, Hono, and Supabase, this project enables users to create campaigns, donate, volunteer, and get personalized campaign recommendations.

⸻

Features
	•	User authentication and signup via Supabase
	•	Campaign creation and management
	•	Donation tracking
	•	Volunteer commitment tracking
	•	AI-powered campaign matching based on user preferences and location
	•	Placeholder campaigns initialized on server startup
	•	Health check endpoint for server monitoring
	•	Full REST API for frontend integration

⸻

Tech Stack
	•	Deno – Runtime for running TypeScript server code
	•	Hono – Lightweight TypeScript web framework
	•	Supabase – Authentication and database services
	•	KV Store – In-memory key-value store for campaigns, donations, and volunteers
	•	TypeScript – Static typing for safer code
  
  
  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
